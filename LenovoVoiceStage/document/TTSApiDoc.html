语音合成接口V1
URL：
http://{ip}/lasf/tts
方法：
POST
格式：
multipart/form-data
头信息：

发起请求时，需要设置http header信息，增加“User-Agent”。User-Agent的值是JSON字符串，格式如：
User-Agent:{platform=[android 4.2.x], name=[com.test.test], version=[1.0.1.130101_xxx]}

参数说明:

名称 格式 描述 取值
platform String 发起请求的平台信息 android 4.2.x
name String 发起请求的应用程序的名称 com.test.test
version String 发起请求的应用程序的版本 1.0.1.130101_xxx
内容信息：

参数说明:

名称 格式 MimeType 描述
param-data String text/plain 将除了下表中所有的参数已URL Encode的形式拼接

参数说明: param-data

名称 英文描述 中文描述 取值
cpf Client Platform 客户端平台 举例：android/windows/linux
dtp Device Type 设备类型 举例：lenovo/ideatv K91/android
ver Client Version 客户端协议版本 举例：1.0.0
did Device Id 设备编号 举例：90:a4:de:ba:41:a5
uid User Id 用户唯一标识 举例：123456789
dev Developer Id 开发者标识 举例：com.lenovo.rt.urc
app Application Name 应用名称 com.lenovo.leos.lestore
stm Start Time 开始时间 举例：1356663021692
key Key 安全密钥 举例：

0ekT/KgQg8uel9nIkTyi1BuBfTU59TKTv

AuENpLV5JiK5zTMirmS8ZG7hDaS1eSYiu
ssm Session Mode 会话模式 枚举：true/false
vdm Domain 垂直领域 枚举：all/ctc/vod/app/was/les
all:全领域识别
ctc:联系人领域(参见相关章节解释)
vod:视频领域
app:应用领域
was:网址查询领域(Web Address Search)
les:乐服务垂直领域
rvr Result Value Range 限定取值 举例：["确定","取消"]
sce Scenario 使用场景 枚举：cmd/iat
ntt Net Type 网络类型 枚举：none/cmnet/cmwap/uninet/uniwap/ctnet/ctwap/wifi/custom
aue Audio Encode 音频编码 格式：算法;等级，举例：aue=speex-wb;7
auf Audio Format 音频格式 枚举：audio/L16;rate=8000 audio/L16;rate=16000
ixid ASR Interaction Id 一次语音交互的标识，由客户端生成与uid一起保证唯一
vcon 文本 合成的文本
返回格式：
Byte方式的语音文件

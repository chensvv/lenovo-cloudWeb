语音识别接口参数说明
URL
https://voice.lenovomm.com/lasf/cloudasr
方法
POST
格式
multipart/form-data
头信息

User-Agent的值是JSON字符串.格式：User-Agent:{platform=[android 4.2.x], name=[com.test.test], version=[1.0.1.130101_xxx]}

参数说明:

名称 格式 取值(样例) 描述
lenovoKey String LENOVO-VOICE-aaabbbccc 账户公钥(右上角开发者信息中获取)
secretKey String kasdjkfsldjsldfjkdkkkde 账户私钥(右上角开发者信息中获取)
channel String cloudasr 来源渠道
内容信息

参数说明:

名称 格式 取值(样例) 描述
scene 字符串 short 场景:长语音long,短语音short,不传默认short
language 字符串 chinese 语言:英文识别english,不传或其它值默认中文
sample 数字 1 通道数:八通道值为8,不传或其他值默认1
audioFormat 字符串 pcm_16000_16bit_sample 语音格式：
pcm_16000_16bit_sample ：16k 16bit 单声道 pcm
pcm_8000_16bit_sample ：8k 16bit 单声道 pcm
alaw_8000_16bit_sample ：8k 16bit 单声道 alaw
ulaw_8000_16bit_sample ：8k 16bit 单声道 ulaw
sessionid 数字 1356663021692 系统时间戳
packageid 数字 1 每个会话数字从1开始迭代
over 数字 1 结束标识:0未结束,1结束
voice-data ByteArray application/i7-voice 语音数据

"asrDur":4254,

"confidence":100,

"desc":"",

"nConf":[100],
多备选置信度，按顺序依次对应

"result":null,
语音处理结果
"status":"success",
结果状态，success:成功，failed:失败

}

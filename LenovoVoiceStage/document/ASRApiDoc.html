API接入指南
语音识别接口参数说明
URL
https://voice.lenovomm.com/lasf/asr
方法
POST
格式
multipart/form-data
头信息

User-Agent的值是JSON字符串.格式:User-Agent:{platform=[android 4.2.x], name=[com.test.test], version=[1.0.1.130101_xxx]}

参数说明:

名称 格式 取值(样例) 描述
platform String android 4.2.x 发起请求的平台信息
name String com.test.test 发起请求的应用程序的名称
version String 1.0.1.130101_xxx 发起请求的应用程序的版本
lenovoKey String LENOVO-VOICE-aaabbbccc 账户公钥(右上角开发者信息中获取)
secretKey String kasdjkfsldjsldfjkdkkkde 账户私钥(右上角开发者信息中获取)
channel String cloudasr 来源渠道
内容信息

参数说明:

名称 格式 内容类型 描述
param-data String text/plain 将下表中所有的参数以URL的格式拼接
voice-data ByteArray application/i7-voice 语音数据

参数说明: param-data

名称 格式 取值(样例) 描述
cpf 字符串 android 客户端平台
dtp 字符串 lenovo 设备类型
ver 字符串 1.0.0 客户端协议版本
did 字符串 90:a4:de:ba:41:a5 设备编号
uid 数字 526 用户唯一标识
dev 字符串 com.lenovo.rt.urc 开发者标识
app 字符串 com.lenovo.leos.lestore 应用名称
stm 数字 1356663021692 开始时间
ssm 布尔类型 false 会话模式
vdm 字符串 all 垂直领域
all:全领域识别
ctc:联系人领域
vod:视频领域
app:应用领域
was:网址查询领域
rvr 字符串 空字符串 限定取值：取值:确定/取消
sce 字符串 cmd 使用场景：cmd:命令模式;iat:听写模式
ntt 字符串 wifi 网络类型
aue 字符串 speex-wb;7 音频编码
auf 字符串 audio/L16;source=outer;rate=16000 音频格式
ixid 数字 1356663021692 会话唯一标识
pidx 数字 1 会话交互内的包的序列号
over 数字 1 是否尾包
0:不是尾包
1:尾包且正常结束
2:尾包,原因客户端主动结束
